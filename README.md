# JobQuest - 就活タスク管理アプリ

## 概要

JobQuestは、就職活動における企業情報、ES締切、選考状況、メモを一元管理するためのWebアプリです。

就活では、複数の企業の締切や選考状況を同時に管理する必要があります。  
その中で、情報がバラバラになったり、締切を見落としたりする課題があると感じ、このアプリを制作しました。

## 主な機能

- 企業情報の追加
- 企業一覧の表示
- 企業情報の編集
- 企業情報の削除
- ブラウザ内へのデータ保存
- 登録企業数の表示
- 今日以降で締切が一番近い企業の表示
- 選考状況ごとの色分け表示

## 入力できる情報

- 企業名
- 業界
- ES締切
- 選考状況
- メモ

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 工夫した点

選考状況ごとに企業カードの色を変えることで、一覧を見たときに現在の選考段階を直感的に把握できるようにしました。

また、今日以降の締切だけを対象に、締切が一番近い企業を表示する機能を実装しました。  
これにより、締切の見落としを防ぎやすくしています。

## 制作を通じて学んだこと

HTMLで画面の構造を作り、CSSで見た目を整え、JavaScriptでデータの追加・編集・削除などの動きを実装する流れを学びました。

また、localStorageを使うことで、ブラウザを閉じてもデータを保存できる仕組みを理解しました。

## 今後追加したい機能

- 締切順での並び替え機能
- 業界ごとの絞り込み機能
- 志望度の登録機能
- 面接日管理機能
- スマホでも見やすいデザインへの改善
- GitHub Pagesでの公開

## アピールポイント

自分自身の就職活動における課題をもとに、企業情報や締切を一元管理できるアプリを制作しました。

単に情報を登録するだけでなく、締切が近い企業の表示や選考状況ごとの色分けを行うことで、実際に使いやすいアプリになるよう工夫しました。
